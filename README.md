# Javul

Javul, **J**ust **A**nother **V**ery **U**seful **L**ibrary (alternatively, **JAV**a **U**tility **L**ibrary), is a collection of handy tools written in pure Java without any external dependencies, ranging from string utilities to a basic arithmetic+boolean expression evaluator. The word "javul" is also a Hungarian verb which means "to improve".
